## 🚀 Tecnologias Utilizadas

| Tecnologia | Versão | Descrição |
|------------|--------|-------------|
| Node.js | 20.x+ | Ambiente de execução JavaScript |
| Express | 4.x | Framework web para Node.js |
| Sequelize | 6.x | ORM para banco de dados |
| SQLite3 | 5.x | Banco de dados relacional |
| JWT | 9.x | Autenticação baseada em token |
| bcryptjs | 2.x | Criptografia de senhas |
| EJS | 3.x | Template engine para views |
| Nodemon | 3.x | Reinicialização automática do servidor |

## 📦 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
